# Petroleum Data Science

This repository showcases my work in applying data analytics and visualization in the oil & gas domain, focusing on production performance and operational insights.

## Project 1: Gas Asset Performance, Decline & Water Risk Analysis  📊 

🔗 **Dashboard**  
https://public.tableau.com/app/profile/aldyth1049/viz/GasAssetPerformanceDeclineWaterRiskAnalysis/GasAssetPerformanceDeclineWaterRiskAnalysis  

📁 **Detailed Analysis**  
https://github.com/aalem90/PetroleumDataScience/tree/main/Tableau-BigDataAnalysis  

**Summary**  
Analysis of ~7M well data to evaluate gas production performance, decline trends, and water risk (WGR).

**Highlights**
- Mature wells show decline + increasing water breakthrough
- Identified opportunities for*optimization, water control, and decommissioning

## Project 2: Well Log Analysis & Lithofacies Classification  🪨 

📁 **Project Detail**  
https://github.com/aalem90/PetroleumDataScience/tree/main/WellLog_Analysis

**Summary**  
Geological data analysis using well logs to identify lithology, facies distribution, and hydrocarbon zones through petrophysical interpretation.

**Highlights**
- Identified clean sand vs shale using Gamma Ray (GR)  
- Detected potential hydrocarbon zones  from high resistivity + porosity correlation  
- Revealed facies heterogeneity and reservoir layering  
- Integrated log visualization for reservoir quality evaluation
